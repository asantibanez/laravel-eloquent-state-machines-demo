## Installation

Clone the repository and install dependencies via Composer. Setup .env and run migrations

```bash
git clone https://github.com/asantibanez/laravel-eloquent-state-machines-demo
cd laravel-eloquent-state-machines
composer install
cp .env.example .env
php artisan migrate
```
